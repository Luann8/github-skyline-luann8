# 🏙️ GitHub Skyline - Luann8

Este repositório contém as visualizações 3D das minhas contribuições no GitHub, geradas através do **GitHub Skyline**. Cada arquivo representa a "cidade" construída pelos meus commits ao longo dos anos.

---

## 📊 O que é o GitHub Skyline?
O GitHub Skyline é uma ferramenta que transforma o seu gráfico de contribuições anual em um modelo 3D estilizado, assemelhando-se a uma linha do horizonte de uma cidade (skyline). Quanto mais contribuições em um dia, mais alto é o "prédio" naquela posição.

## 📁 Arquivos no Repositório

## 🛠️ Como Visualizar
Se você não tem uma impressora 3D, pode visualizar os arquivos de várias formas:
1. **Visualizador Nativo do GitHub:** O próprio GitHub consegue renderizar arquivos `.stl` diretamente no navegador. Basta clicar no arquivo acima.
2. **Online:** Use sites como o [ViewSTL](https://www.viewstl.com/).
3. **Software:** Utilize o Blender, Windows 3D Viewer ou Ultimaker Cura.
